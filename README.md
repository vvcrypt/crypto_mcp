# Crypto Data MCP Server
A Model Context Protocol (MCP) server providing LLMs with access to cryptocurrency market data from multiple exchanges.

### What is this?
This MCP server acts as a bridge between large language models (like Claude) and cryptocurrency exchanges. It provides a unified interface to query market dataâ€”no API keys, no rate limit headaches, no exchange-specific quirks.

### Supported Data: 
> Open Interest; Current and historical OI for futures

> Funding Rates: Current and historical funding rates

> Ticker: 24h price, volume, change statistics

>Klines/OHLCV

>Historical candlestick data

>Mark Price

>Current mark price for futures

### Supported Exchanges
> Binance (USDT-M Futures)
> Bybit (planned)
> OKX (planned)
> Hyperliquid (planned)
